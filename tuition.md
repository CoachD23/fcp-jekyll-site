<div class="program-offers fade-in" style="margin-top: 3rem;">
  <div class="staff-section-header">
    <span class="section-label">FAQ</span>
    <h2>Frequently Asked Questions</h2>
    <div class="gold-rule"></div>
  </div>
  <div class="faq-grid" style="margin-top: 2rem;">
    <div class="faq-card fade-in">
      <h3>What grade levels does the National High School Program serve?</h3>
      <p>Our National High School Program is designed for student-athletes in grades 9–12. We also accept post-graduate (PG) year students who want an additional year of elite development before college. All athletes must be within standard NCAA eligibility windows.</p>
    </div>
    <div class="faq-card fade-in">
      <h3>Is the academic program accredited?</h3>
      <p>Yes. Our academic curriculum is delivered through an accredited online program that meets state and NCAA eligibility requirements. Students complete coursework at their own pace with dedicated academic support, and transcripts are issued upon request for college applications and recruiting.</p>
    </div>
    <div class="faq-card fade-in">
      <h3>Does tuition include housing and meals?</h3>
      <p>Yes — tuition covers supervised athlete housing and meals. Student-athletes live together in a structured, team-focused residential environment on Florida's Emerald Coast. All meals are provided through our approved dining program, with nutrition guidance included in the Performance component of our training.</p>
    </div>
    <div class="faq-card fade-in">
      <h3>What does "optional dual enrollment" mean?</h3>
      <p>Qualified high school juniors and seniors may enroll in college-level courses simultaneously with their FCP curriculum, earning transferable college credits before graduation. Dual enrollment is offered as an optional add-on for eligible students and is coordinated through our academic team.</p>
    </div>
    <div class="faq-card fade-in">
      <h3>How does FCP support the recruiting process?</h3>
      <p>Every enrolled athlete receives a personalized recruiting plan developed with our staff. This includes game film production and distribution, exposure at national tournaments, direct outreach to college coaches, and ongoing guidance on NCAA compliance and the recruiting timeline.</p>
    </div>
    <div class="faq-card fade-in">
      <h3>Do international athletes need health insurance?</h3>
      <p>Yes. All international student-athletes are required to carry valid health insurance for the duration of their enrollment. FCP can provide guidance on approved international student health plans if needed. Domestic athletes are encouraged to maintain their existing coverage throughout the program year.</p>
    </div>
  </div>
</div>

<script>
(function () {
  var form = document.getElementById("fcp-payment-form");
  if (!form) return;

  form.addEventListener("submit", function (e) {
    e.preventDefault();

    var nameEl = document.getElementById("pay-name");
    var purposeEl = document.getElementById("pay-purpose");
    var amountEl = document.getElementById("pay-amount");
    var submitBtn = document.getElementById("pay-submit");
    var loadingEl = document.getElementById("payment-loading");
    var errorEl = document.getElementById("payment-error");

    var name = (nameEl.value || "").trim();
    var purpose = purposeEl.value;
    var amount = parseFloat(amountEl.value);

    // Reset error
    errorEl.style.display = "none";

    // Validate
    if (!name) { showPayError("Please enter the participant name."); nameEl.focus(); return; }
    if (!purpose) { showPayError("Please select the purpose of deposit."); purposeEl.focus(); return; }
    if (isNaN(amount) || amount < 10) { showPayError("Minimum payment is $10.00."); amountEl.focus(); return; }
    if (amount > 50000) { showPayError("For payments over $50,000, please call 850-961-2323."); return; }

    // Show loading
    form.style.display = "none";
    loadingEl.style.display = "block";
    submitBtn.disabled = true;

    fetch("/.netlify/functions/create-payment", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        participantName: name,
        purpose: purpose,
        amount: amount.toFixed(2)
      })
    })
      .then(function (res) { return res.json(); })
      .then(function (data) {
        if (data.token) {
          // Redirect to Authorize.net hosted payment page
          document.getElementById("authnet-token").value = data.token;
          document.getElementById("authnet-redirect").submit();
        } else {
          showPayError(data.error || "Unable to initialize payment. Please try again.");
          resetPayForm();
        }
      })
      .catch(function () {
        showPayError("Connection error. Please check your internet and try again.");
        resetPayForm();
      });
  });

  function showPayError(msg) {
    var el = document.getElementById("payment-error");
    el.textContent = msg;
    el.style.display = "block";
  }

  function resetPayForm() {
    document.getElementById("payment-loading").style.display = "none";
    document.getElementById("fcp-payment-form").style.display = "block";
    document.getElementById("pay-submit").disabled = false;
  }
})();
</script>
